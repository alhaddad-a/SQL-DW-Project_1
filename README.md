# SQL Data Warehouse Project

## Overview

This repository contains the source code and resources for the **SQL Data Warehouse (SQL-DW) Project**. The project demonstrates best practices for designing, building, and managing a scalable data warehouse using SQL Server technologies.

## Features

- Structured schema for analytical workloads
- ETL pipelines using stored procedures and scripts
- Sample data sets for testing and demonstration
- Documentation and scripts for deployment

## Getting Started

### Prerequisites

- Windows 10/11 or a compatible environment
- SQL Server 2019 or later (Express, Developer, or Enterprise edition)
- SQL Server Management Studio (SSMS) or Azure Data Studio

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/SQL-DW-Project_1.git
   cd SQL-DW-Project_1
   ```
2. Open the solution in SSMS or Azure Data Studio.
3. Run the `setup.sql` script to create the database, tables, and sample data.
4. Review the `etl/` folder for ETL scripts and adjust connection strings as needed.

### Usage

- Execute the provided stored procedures to load and transform data.
- Use the sample queries in `queries/` to explore the data warehouse.
- Extend the schema and ETL processes to fit your own data sources.
